# Scaffold Discovery - MediFlow

Atividade da disciplina Projeto Integrado de Aplicações do curso Desenvolvimento Full Stack, voltado ao cenário comum em clínicas médicas, com tudo elaborei os markdown do projeto chamado MediFlow.

## Sobre o Projeto

Este repositório contém os artefatos da fase de **Discovery e Refinamento** para o projeto **MediFlow**, desenvolvido como parte de um desafio prático da disciplina. 

O **MediFlow** é uma plataforma de pré-atendimento inteligente projetada para clínicas populares. O sistema automatiza a confirmação de consultas via integração com a API do WhatsApp e realiza uma pré-triagem digital do paciente, reduzindo drasticamente as taxas de *no-show* (faltas) e otimizando o tempo de espera na recepção.

### Objetivos do Desafio
- **Clareza de escopo e redução de ambiguidade:** Definir o problema antes da solução.
- **Consistência:** Alinhamento total entre requisitos (PRD), design de interface e arquitetura técnica.
- **Desenvolvimento Assistido por IA:** Uso eficiente de modelos de Inteligência Artificial (Prompt Engineering) como copilotos em todo o fluxo de discovery.
- **Single Source of Truth:** Utilização do GitHub como repositório central e Markdown como "língua franca" da documentação.
